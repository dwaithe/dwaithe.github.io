>Altering Interactive Cue Card pronunciation

The above Interactive Cue Card has been linked to mp3 clips recorded using Standard Southern British English. If a user would like to use the Interactive Cue Card as a tool for teaching, but would like to change the sound clips so they reflect a local dialect, this is possible by following the below steps, which require some level of computer skill to perform:

  1. Download the interactive zip file by right clicking this link, and clicking "Save Target As". Save the file to a convenient location such as the desktop.
  2. Locate the zip file where it was saved (i.e. the desktop) and unzip the contents to again a convenient location.
  3. The interactive cue card can be used by double clicking the interactive-cue-card.exe file within the folder created by the unzipping process. The interactive cue card functions in the same way as the web based version above.
  4. To alter the pronunciation, "mp3" sound files must be recorded for each of the individual sounds that are being replaced. This can be achieved by using any commercial mp3 recording software and a microphone.
  5. The default "mp3" sound files for each of the MPC phonic sounds are located within the sound folder, which can be found within the root unzipped folder. The sound file names correspond to phonic sounds as follows:
