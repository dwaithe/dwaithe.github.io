Interactive Cue Card

Fundamentals of Minimal Phonic Cues

Below is the Interactive Cue Card. It has been designed to illustrate a suggested method for using the Cue Card to teach the sounds of the English language with MPC. Altering Interactive Cue Card pronunciation

The above Interactive Cue Card has been linked to mp3 clips recorded using Standard Southern British English. If a user would like to use the Interactive Cue Card as a tool for teaching, but would like to change the sound clips, so they reflect a local dialect, this is possible by following the below steps, which require some level of computer skill to perform:

Using your browser, save this web page to a directory on your computer.

  1. Create tow folders within the directory you have and name them 'sound' and 'images'.
  2. Within the folder 'images' create a folder called 'Cue card'.
  3. Download this image (colour-medium) and place it in the folder named 'Cue card'
  4. Download this image (mono-medium) and place it in the folder named 'Cue card'
  5. To alter the pronunciation, "mp3" sound files must be recorded for each of the individual sounds that are being replaced. This can be achieved by using any commercial mp3 recording software and a microphone.
